Заказать поверку

Правила проведения стандартной поверки счетчика воды

  1. Вызвать специалиста из Горводоканала для снятия пломбы (самостоятельно срывать пломбу запрещается Водным кодексом) (2-3 дня)
  2. Вызвать специалиста из Горводоканала для снятия пломбы (самостоятельно срывать пломбу запрещается Водным кодексом) (2-3 дня)
  3. Вызвать водопроводсчика для снятия счетчика воды, либо самостоятельно снять счетчик (1 день)
  4. Сдать счетчик на поверку в лабораторию (до 1 недели)
  5. Вызвать водопроводсчика для установки счетчика воды, либо самостоятельно установить счетчик (1 день)
  6. Вызвать специалиста из Горводоканала для пломбировки счетчика (2-3 дня)
Итого около 2-х недель
